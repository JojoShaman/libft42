# LIBFT

This project is part of the 42 curriculum, created by **srosu**.

## Goals

Before starting larger projects, having a set of basic tools is essential. The first project in the curriculum is `Libft`, a custom C library containing commonly used functions that will be reused in future projects.

## Part 1 – Libc Functions

- `ft_isalpha` → check letter  
- `ft_isdigit` → check digit  
- `ft_isalnum` → check alphanumeric  
- `ft_isascii` → check ASCII  
- `ft_isprint` → check printable  
- `ft_strlen` → string length  
- `ft_memset` → fill memory  
- `ft_bzero` → zero memory  
- `ft_memcpy` → copy memory  
- `ft_memmove` → safe copy  
- `ft_strlcpy` → copy string  
- `ft_strlcat` → concat string  
- `ft_toupper` → to uppercase  
- `ft_tolower` → to lowercase  
- `ft_strchr` → find char  
- `ft_strrchr` → find char (last)  
- `ft_strncmp` → compare strings  
- `ft_memchr` → find in memory  
- `ft_memcmp` → compare memory  
- `ft_strnstr` → find substring  
- `ft_atoi` → string to int  
- `ft_calloc` → allocate zeroed  
- `ft_strdup` → duplicate string  

## Part 2 – Additional Functions

- `ft_substr` → extract substring  
- `ft_strjoin` → join strings  
- `ft_strtrim` → trim chars  
- `ft_split` → split string  
- `ft_itoa` → int to string  
- `ft_strmapi` → map string  
- `ft_striteri` → iterate string  
- `ft_putchar_fd` → print char fd  
- `ft_putstr_fd` → print string fd  
- `ft_putendl_fd` → print line fd  
- `ft_putnbr_fd` → print number fd  

## Part 3 – Linked List Functions

- `ft_lstnew` → create node  
- `ft_lstadd_front` → add front  
- `ft_lstadd_back` → add back  
- `ft_lstsize` → list size  
- `ft_lstlast` → last node  
- `ft_lstdelone` → delete node  
- `ft_lstclear` → clear list  
- `ft_lstiter` → iterate list  
- `ft_lstmap` → map list  

## Resources

This project was developed using `Linux man pages` and the `ISO/IEC 9899:2024 C standard` as primary references. Additional explanations and examples were consulted through YouTube tutorials, particularly `mycodeschool`, as well as forums such as `Stack Overflow`. AI tools, including `ChatGPT` and `Claude`, were explicitly used to better understand certain functions, explore alternative implementations, and optimize the overall coding process.
